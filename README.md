# 🔍 Auto_Recon - Automated Pentesting Reconnaissance

## 📊 Overview
Bash automation script that streamlines initial enumeration during penetration testing, running Nmap, Nikto, and enum4linux sequentially.

## 🚀 Features
- Automated port scanning with Nmap
- Web vulnerability scanning with Nikto
- SMB enumeration with enum4linux
- Structured output files

## 🔧 Usage
```bash
chmod +x auto_recon.sh
./auto_recon.sh 192.168.1.100
